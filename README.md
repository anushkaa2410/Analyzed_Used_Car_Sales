# Analyzed_Used_Car_Sales
Exploratory Data Analysis (EDA) of a used car sales dataset using Python. The project analyzes vehicle prices, mileage, condition, brands, and body types through data cleaning and visualizations to identify key factors influencing used car prices.

# Used Car Sales Analysis

## Overview

This project performs Exploratory Data Analysis (EDA) on a Used Car Sales dataset using Python. The objective is to analyze the factors affecting used car prices by exploring relationships between vehicle condition, mileage, manufacturer, body type, trim level, and selling price through various visualizations.

The project demonstrates the complete data analysis workflow, including data cleaning, preprocessing, exploratory analysis, and interpretation of insights.

## Objectives

- Clean and preprocess the dataset.
- Analyze the distribution of used car prices.
- Identify the top car manufacturers in the dataset.
- Study the relationship between vehicle mileage and selling price.
- Analyze how vehicle condition affects resale value.
- Compare selling prices across different manufacturers and trim levels.
- Generate meaningful business insights through visualizations.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Exploratory Data Analysis

The project includes the following analyses:

- Data Cleaning & Preprocessing
- Pair Plot Analysis
- Correlation Heatmap
- Top Car Manufacturers
- Odometer vs Selling Price Analysis
- Vehicle Condition Distribution
- Average Selling Price by Condition
- Ford F-150 Price Distribution
- Trim Level vs Selling Price
- Condition vs Selling Price

## Key Insights

- Vehicle mileage has a negative impact on selling price, with higher mileage generally leading to lower resale values.
- Premium manufacturers such as BMW maintain higher resale prices than many mainstream brands.
- Vehicle condition plays a significant role in determining resale value.
- Popular manufacturers dominate the used car market, indicating higher customer demand.
- Premium trim levels command significantly higher prices than standard trims.
- Correlation analysis reveals meaningful relationships among numerical variables that can support predictive modeling.

## Final Conclusion

This project successfully explored the factors influencing used car prices through comprehensive Exploratory Data Analysis (EDA). The analysis demonstrated that mileage, vehicle condition, manufacturer, and trim level are among the most important variables affecting resale value.

Visualizations provided valuable insights into market trends, pricing behavior, and brand performance, enabling a better understanding of the used car market. The project showcases practical data analysis skills including data preprocessing, visualization, statistical interpretation, and business insight generation using Python.

This analysis can serve as a strong foundation for future machine learning models aimed at predicting used car prices.
